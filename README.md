# Django Web Project

## Setup

In Linux,

```bash
$ # setup virtual env for python
$ sudo apt install python3
$ pip install virtualenv
$ mkdir djproj && cd djproj
$ virtualenv .
$ source bin/activate
$
$ # clone repo and activate virtual env
$ git clone url/of/Django-Web
$ cd Django-Web

$ pip install django
```

## Run

In Linux.

```bash
$ cd Django-Web
$ source bin/activate
$ python3 manage.py runserver
```
